# CSV Data Processing System

## Features
- Reads employee CSV data
- Calculates department averages
- Finds highest paid employees
- Adds bonus salary column
- Filters employees above average
- Logs processing activity

## Run Program
python csv_processor.py

## Files
- employees.csv (input)
- updated_employees.csv (output)
- processor.log (logs)

## Requirements
Python 3.x
